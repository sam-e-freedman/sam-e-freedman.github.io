Featured Thesis Project

Modeling Post-Wildfire Carbon Recovery Using LiDAR, Remote Sensing, and Machine Learning

My M.S. thesis in Geographic Information Science investigates post-fire ecosystem recovery and carbon dynamics in California forests. The project compares the CZU Lightning Complex and Dixie Fire using airborne LiDAR, GEDI, satellite imagery, burn severity data, vegetation layers, and machine learning models.

This work integrates GIS, raster analysis, remote sensing, statistical modeling, and ecological interpretation to understand how forest carbon changes after wildfire across different California forest systems.
